Redesigning Craigslist

View the responsive page here

The Task

A partner and I wanted to redesign the Craigslist website to be cleaner and easier to navigate, as well as responsive to different devices and screen sizes.

The Process

We first created wireframes of a few of the website's new pages. Once satisfied with the design, we drew out a diagram of how elements should be change in response to changing screen sizes. We went on to create a high-fidelity mockup of the site on Proto.io, and then I created the responsive "add listing" page using HTML and CSS.

Feedback

We recieved the feedback that we should make functionality more central to the design of the page, with the most likely tasks being the easiest to do (for example, the toolbar allowing users to change between different continents would likely not be used very frequently, while one navigating from home to listings would).

Takeaways

This project was a great opportunity to learn about responsive design. However, my Professor pointed out that Craigslist may also intentionally maintain a haphazard appearance because it allows users so much freedom. They do not want it to seem like the products listed are verified or even safe. I realized that the goal is not to make every interface look maximally clean and polished, but rather to clearly communicate what the user can do, while keeping their best interest in mind, and helping the company achieve its goals.
